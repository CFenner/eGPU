### Disable Auto Update

https://wccftech.com/how-to/how-to-disable-windows-10-automatic-updates/

- press `Windows` + `R`
- type `services.msc`
- search service `Windows Update`
- stop the service
- open service properties
- set `startup type` to `disable`
